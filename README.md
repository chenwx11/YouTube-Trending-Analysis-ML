# YouTube-Trending-Analysis-ML
# YouTube 热门视频趋势分析与流量预测

本项目通过Python对YouTube全球热门视频数据进行深度挖掘，涵盖数据集成、清洗、文本挖掘及机器学习建模。

## 🛠️ 技术栈
- 数据处理：Pandas, JSON
- 可视化：Seaborn, Matplotlib, WordCloud
- 机器学习：XGBoost (Regression)

## 📈 核心成果
- **数据处理**：完成了异构 JSON 与 CSV 数据的标准化关联。
- **流量洞察**：识别出不同内容赛道的爆火周期差异。
- **精准预测**：构建流量回归模型，R² 达到 0.80，识别出“点赞数”为第一核心驱动因子。

## 📂 目录说明
- `/scripts`: 数据清洗与建模源码
- `/results`: 核心分析图表
- `YouTube_Analysis_Report.pdf`: 完整版分析报告
